## Python History Overview

- **Creator:** Guido van Rossum
- **Year Created:** Development began December 26, 1989; first public release 0.9.0 in February 1991
- **Design Philosophy:** Emphasis on code readability, simplicity, and the principle "There should be one – and preferably only one – obvious way to do it". Explicit is better than implicit.

### Key Milestones
- 1991: Python 0.9.0 released
- 2000: Python 2.0 introduced (list comprehensions, garbage collection)
- 2008: Python 3.0 released (breaking changes, print function, Unicode handling)
- 2021: Python 3.10 released (pattern matching)
- 2023: Python 3.12 released (improved error messages, performance)
- 2026: Python 3.14.3 (latest stable as of Feb 2026)
