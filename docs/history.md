# Python History

**Creator:** Guido van Rossum

**Year Created:** 1991 (first appeared on 20 February 1991)

**Design Philosophy:** Emphasizes code readability with significant indentation; principles include "Beautiful is better than ugly", "Explicit is better than implicit", "Simple is better than complex", "Readability counts". Includes a "batteries‑included" standard library.

## Key Milestones
- **1991:** First release (Python 0.9.0).
- **2000:** Python 2.0 released, introducing list comprehensions and a garbage collector.
- **2008:** Python 3.0 released, breaking backward compatibility (print function, integer division, Unicode strings).
- **2023:** Python 3.12 released with pattern matching enhancements.
- **2025:** Python 3.13 released.
- **2026:** Python 3.14.3 released (current stable version).
